# Dr. Krishnan Chandrasekharan | Professional Portfolio Hub

A production-grade, multi-page professional portfolio website designed for **Dr. Krishnan Chandrasekharan**, Leadership Strategist, Executive Coach, and Founder & CEO of Learning Without Walls.

## 🚀 Key Features
- **Corporate Branding**: Premium Blue (#0d6efd) and White aesthetic.
- **Deep Bio Integration**: Full profile history covering 25+ years and 150k+ trainees.
- **Strict Media Integrity**: Uses only the provided high-quality Unsplash and YouTube assets.
- **Responsive Architecture**: Built with Bootstrap 5.3 for mobile-first visibility.
- **Interactive Experience**: AOS animations, smooth scrolling, and lightbox visuals.
- **Multi-Page Navigation**: Dedicated pages for About, Gallery, Videos, Blog, and Contact.
